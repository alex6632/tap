# tap
MEAN exercise

Requirements
------------
* mongodb
* node.js v6.0 or later
* angular-cli v1.0.0 or later

Install
-------
On first install, run
```
npm install
```

1/ Run server with nodemon
```
npm run start
```
or just
```
nodemon
```

2/ Run mongodb
```
mongod
```
If you have access problem,
```
sudo mongod
```

3/ Run angular
```
ng serve
```
